ISAAC Software

About

The Integrated System for Autonomous and Adaptive Caretaking project is developing technology for combining robots inside a spacecraft with vehicle infrastructure subsystems, to form an integrated autonomous system. The system validates its technology through demonstrations using the Astrobee free-flier robots, existing robots onboard the International Space Station as analogs for future robots that will be developed for the Gateway.

The ISAAC project actively develops in a variety of repos hosted at Ames and JSC.

This isaac repo serves as a master for integrating an end-to-end demo that draws on code from the other repos as well as directly including a significant amount of the ISAAC code, mostly relating to the Astrobee robot.

System requirements

The isaac repo depends on the astrobee repo, therefore it inherits the same system requirements. You must use Ubuntu 16.04 to 20.04 64-bit. When running in simulation, certain Gazebo plugins require appropriate graphics drivers.

Usage

There are two main ways to install and run isaac:

  1. For development: Build the isaac code on your native Ubuntu OS (or inside a normal virtual machine) in a way that makes it convenient for you to edit and incrementally recompile.
  2. For demo: For the ISAAC integrated demo, many ISAAC repos are checked out, built, and run in a distributed fashion across multiple docker containers. The isaac code itself is built and run inside one of these containers. Note that the in-docker build is managed by the Dockerfile and completely separate from any build in your native OS, and you don't need to install for development prior to installing for demo.
